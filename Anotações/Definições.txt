1) O que é o PHP:
-> Linguagem de programação que roda diretamente no servidor
-> Linguagem usada para Web
-> Linguagem gratuita
-> Pode-se criar muitos recursos e possui diversos Frameworks e bibliotecas
-> Rápida execução
exemplo de linha de código:
<?php echo "Olá mundo!"?>

2) Front-End vs Back-End
-> Front-End é a parte da "frente". Visual que nosso navegador executa. (HTML, CSS, JavaScript)
-> Back-End é a parte de trás. O que executa no servidor (ex: PHP, Database...)

3) Servidor APACHE
-> É um servidor Web e é usado para interpretar código PHP, usa protocolo http, é  seguro e estável.
-> Lógica do Apache: (É como um garçom)
	L> quando acessamos o site ou sistema, enviamos a requisição para o servidor
	   Começa a executar e ler o código
	L> manda de volta para o navegador (Front-End)

4) Para usar o PHP o ideal é termos um bom navegador, Editor de código, um servidor APACHE e um servidor
de banco de dados
